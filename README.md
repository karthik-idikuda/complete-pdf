# Complete PDF

Free online PDF editor with 50+ tools for merging, splitting, compressing, converting, and annotating PDF documents.

## Features

- Merge multiple PDFs into one
- Split PDF by page ranges
- Compress PDF file size
- Convert between PDF and image formats
- Add text annotations and highlights
- Page rotation and reordering

## Tech Stack

TypeScript, React, PDF.js, Node.js

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/karthik-idikuda/complete-pdf.git
cd complete-pdf
npm install
```

### Usage

```bash
npm run dev
```
